Birthday Reminder Bot
Telegram-бот для напоминания о днях рождения друзей. Пользователь может сохранять имена и даты рождения, а бот ежедневно в 9:00 утра присылает уведомление, если сегодня у кого-то из списка день рождения.

Функционал
Добавление записи о друге: имя и дата рождения (формат ДД.ММ.ГГГГ)
Просмотр списка всех сохранённых друзей
Удаление записи по её идентификатору
Ежедневная автоматическая проверка дней рождений в 9:00
Отправка пользователю списка именинников на текущий день

Технологии
Python 3.8+
python-telegram-bot (асинхронная версия 20.x) — взаимодействие с Telegram API
APScheduler — планировщик фоновых задач
SQLite — база данных для хранения записей пользователей

Установка и запуск
Клонируйте репозиторий или создайте папку проекта со всеми файлами.
Установите зависимости:
pip install python-telegram-bot apscheduler
Получите токен бота у @BotFather в Telegram.
В файле config.py замените "ВАШ_ТОКЕН_ЗДЕСЬ" на полученный токен.

Запустите бота:
python main.py
После запуска бот будет работать в режиме polling. База данных birthdays.db создаётся автоматически при первом запуске.

Команды бота
/start — приветственное сообщение и краткая справка
/help — подробная справка по командам
/add <имя> <ДД.ММ.ГГГГ> — добавить друга (имя может состоять из нескольких слов, дата строго в указанном формате)
/list — показать список всех сохранённых друзей (с идентификаторами)
/remove <id> — удалить запись по её идентификатору (id виден в списке)

Структура проекта
birthday_bot/
├── main.py          # точка входа, настройка приложения и планировщика
├── config.py        # токен бота
├── db.py            # функции работы с базой данных SQLite
├── handlers.py      # обработчики команд пользователя
├── scheduler.py     # планировщик и функция ежедневной проверки
├── utils.py         # вспомогательные функции (парсинг и форматирование даты)
└── birthdays.db     # файл базы данных (создаётся автоматически)
Описание работы
Пользователь отправляет команду /add с именем и датой рождения. Бот проверяет корректность даты и сохраняет запись в БД.

По команде /list пользователь видит все свои записи с номерами (id), которые можно использовать для удаления.
Команда /remove удаляет выбранную запись, если она принадлежит данному пользователю.
Планировщик запускается вместе с ботом и каждый день в 9:00 утра выполняет функцию check_birthdays. Она:
получает из БД всех друзей всех пользователей;
сравнивает текущие день и месяц с сохранёнными датами;
для каждого пользователя, у которого есть именинники, отправляет сообщение со списком имён.

Примечания
Дата вводится строго в формате ДД.ММ.ГГГГ (например, 15.05.1990). При неверном формате бот сообщит об ошибке.
В базе данных дата хранится в формате ISO (YYYY-MM-DD) для удобства сравнения.
Планировщик сбрасывается при перезапуске бота: если бот был остановлен во время запланированной проверки, уведомления в этот день не отправятся.
Для работы необходим доступ в интернет и запущенный Python-скрипт.
